Graphical models provide a symbolic approach to factoring joint probability distributions. Representing a probability distribution within the graphical model formalism involves making certain CI assumptions, assumptions which are embedded in the structure of the graph. From the graphical structure other independence relations can be derived, reflecting the fact that certain factorizations of joint probability distributions imply other factorizations. The key advantage of the graphical approach is that such factorizations can be read off from the graph via simple graph search algorithms.

Missing variables in the local CDFs correspond to missing edges in the underlying graph. Three canonical three-node graphs

(a) Cascade: X→Y→ZX\rightarrow Y\rightarrow ZX→Y→Z. The missing edge in this graph corresponds to the CI statement X⊥Z∣YX\perp Z\vert YX⊥Z∣Y. Moreover, there are no other CIs associated with this graph. Interpretation: a simple Markov chain — the past is independent of the future given the present.

Proof:

p(z∣x,y)=p(x,y,z)p(x,y)=p(x)p(y∣x)p(z∣y)p(x)p(y∣x)=p(z∣y)\begin{align*} p(z|x, y)&=\frac{p(x, y, z)}{p(x, y)} \\ &=\frac{p(x)p(y|x)p(z|y)}{p(x)p(y|x)} \\ &=p(z|y) \end{align*}p(z∣x,y)​=p(x,y)p(x,y,z)​=p(x)p(y∣x)p(x)p(y∣x)p(z∣y)​=p(z∣y)​

(b) Common parent: X←Y→ZX\leftarrow Y\rightarrow ZX←Y→Z. The missing edge in this graph corresponds to the CI statement X⊥Z∣YX\perp Z\vert YX⊥Z∣Y and no other CIs are associated with this graph. Interpretation: hidden variable YYY explains all of the observed dependence between XXX and ZZZ.

Proof:

p(x,z∣y)=p(y)p(x∣y)p(z∣y)p(y)=p(x∣y)p(z∣y)\begin{align*} p(x, z|y)&=\frac{p(y)p(x|y)p(z|y)}{p(y)} \\ &=p(x|y)p(z|y) \end{align*}p(x,z∣y)​=p(y)p(y)p(x∣y)p(z∣y)​=p(x∣y)p(z∣y)​

(c) V-structure (aka explaining away): X→Y←ZX\rightarrow Y\leftarrow ZX→Y←Z. The missing edge in this graph corresponds to the marginal independence statement X⊥ZX\perp ZX⊥Z. Interpretation: observing YYY explains away.

Graph separation

Let AAA, BBB, CCC be three sets of nodes in a Bayesian network GGG. We say that AAA and BBB are ddd-separated given CCC if AAA and BBB are not connected by an active path. An undirected path in GGG is called active given observed variables CCC if for every consecutive triple of variables X,Y,ZX, Y, ZX,Y,Z on the path, one of the following holds:

  • head-to-tail X←Y←ZX\leftarrow Y\leftarrow ZX←Y←Z or X→Y→ZX\rightarrow Y\rightarrow ZX→Y→Z, and YYY is unobserved Y∉CY\notin CY∈/C
  • tail-to-tail X←Y→ZX\leftarrow Y\rightarrow ZX←Y→Z, and YYY is unobserved Y∉CY\notin CY∈/C
  • head-to-head X→Y←ZX\rightarrow Y\leftarrow ZX→Y←Z, and YYY or any of its descendants are observed.

In the following example, X1X_1X1​ and X6X_6X6​ are ddd-separated given X2X_2X2​, X3X_3X3​.

However, X2X_2X2​, X3X_3X3​ are not ddd-separated given X1X_1X1​, X6X_6X6​ due to an active path passing through the V-structure created when X6X_6X6​ is observed.

Markov blanket (MB)

We define the Markov blanket UUU of a variable XXX as the minimal set of nodes s.t. XXX is independent of the rest of the graph if UUU is observed.

The MB of a node XiX_iXi​ comprises the set of its parents, children, and co-parents. Conditional distributional of XiX_iXi​, conditioend on all remaining variables in the graph, is dependent only the variables in the MB.

Independence map

Denote I(p)I(p)I(p) as the set of all independencies that hold for a joint distribution ppp, and I(G)={(X⊥Y∣Z)∣X,Yare d-sep given Z}I(G)=\{(X\perp Y\vert Z)|X, Y \text{are }d\text{-sep given }Z\}I(G)={(X⊥Y∣Z)∣X,Yare d-sep given Z}.

Fact: If ppp factorizes over GGG, then I(G)⊆I(p)I(G)\subseteq I(p)I(G)⊆I(p). We say that GGG is an III-map for ppp.

In other words, all the independencies encoded in GGG are sound: variables that are ddd-separated in GGG are truly independent in ppp. However, the converse is not true: a distribution may factorize over GGG, yet have independencies that are not captured in GGG.

Question: can directed graphs express all the independencies of any distribution ppp i.e. given a distribution ppp, can we construct a graph GGG s.t. I(G)=I(p)I(G)=I(p)I(G)=I(p)?

First, note that it is easy to construct a GGG s.t. I(G)⊆I(p)I(G)\subseteq I(p)I(G)⊆I(p) e.g. a fully connected DAG which implies I(G)=∅I(G)=\emptyI(G)=∅. Additionally, it is possible to find a minimal III-map GGG for ppp: start with a fully connected graph and remove edges until GGG is no longer an III-map following the natural topolotical ordering.

However, it is not true that any probability distribution ppp always admits a perfect map GGG for which I(p)=I(G)I(p)=I(G)I(p)=I(G).

E.g. Consider distribution ppp over three variables XXX, YYY, ZZZ where X,Y∼Ber(0.5)X, Y\sim \text{Ber}(0.5)X,Y∼Ber(0.5) and Z=XxorYZ=X\text{xor} YZ=XxorY. We can derive that X⊥Y,Z⊥Y,X⊥Z∈I(p){X\perp Y, Z\perp Y, X\perp Z}\in I(p)X⊥Y,Z⊥Y,X⊥Z∈I(p) but Z⊥Y,X∉I(p)Z\perp{Y, X}\notin I(p)Z⊥Y,X∈/I(p). Thus, X→Z←YX\rightarrow Z\leftarrow YX→Z←Y is an III-map for ppp, but non of the three-node graph structures that we discussed perfectly describes I(p)I(p)I(p), and hence this distribution doesn't have a perfect map.

Perfect maps are not unique when they exist. E.g. X←YX\leftarrow YX←Y and X→YX\rightarrow YX→Y encode the same independencies but form different graphs. More generally, we say that two Bayes nets G1G_1G1​, G2G_2G2​ are III-equivalent if they encode the same dependencies i.e. I(G1)=I(G2)I(G_1)=I(G_2)I(G1​)=I(G2​).
